# Convert Markdown to PowerPoint

Ce projet permet de convertir un fichier Markdown en une présentation PowerPoint.

## Prérequis

- Python 3.x
- Les bibliothèques Python suivantes :
  - `python-pptx`
  - `markdown`
  - `beautifulsoup4`
